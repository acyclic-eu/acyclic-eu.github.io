Hybrids Components Examples

This page demonstrates the available web components built with the Hybrids library (v9.1.18).

Simple Counter Component

A basic counter component that increments a value when clicked.

Basic Usage

<simple-counter></simple-counter>

Custom Starting Value

const counter = document.createElement('simple-counter');
counter.count = 10; // Set custom starting value
document.querySelector('#container').appendChild(counter);

Tag Toggle Component

A toggle component for filtering or selecting categories. Features hover tooltips and toggle state.

Basic Usage

<tag-toggle name="JavaScript" description="A programming language for the web"></tag-toggle>

Pre-checked State

<tag-toggle name="TypeScript" description="JavaScript with syntax for types" checked></tag-toggle>

Event Handling Demo

Status: unchecked
const tag = document.querySelector('#demo-tag');
tag.addEventListener('change', (e) => {
document.querySelector('#output').textContent = `Status: ${e.target.checked ? 'checked' : 'unchecked'}`;
});
